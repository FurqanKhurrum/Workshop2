# Vector and Matrix Math Library

A custom 3D math library implementation in C# with OpenTK comparison.

## Features
- Vector3D class with operations:
  - Addition, Subtraction
  - Dot Product, Cross Product
  - Magnitude, Normalization
  
- Matrix4x4 class with operations:
  - Identity, Scale, Rotation (X/Y/Z axes)
  - Translation, Matrix Multiplication
  - Vector/Point transformations

## Requirements
- .NET Framework/Core
- OpenTK NuGet package

## Usage
Run the console application to see demonstrations of all operations.

## Validation
Results are validated against OpenTK's math library for accuracy.
